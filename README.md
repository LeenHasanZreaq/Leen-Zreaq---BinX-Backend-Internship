# C# Basics - Practice 1

This project contains my first C# practice program. It demonstrates the basic syntax and fundamental programming concepts in C#.

## What I Learned

### Variables and Data Types
- `char`
- `double`
- `float`
- `int`
- `long`
- `short`
- `string`
- `bool`

### Console Input and Output
- Display text using `Console.WriteLine()`
- Read user input using `Console.ReadLine()`

### Conditional Statements
- `if`
- `else if`
- `else`

### Switch Statement
- Using `switch` and `case` to execute different code blocks.

### Loops
- `for` loop
- `while` loop
- `do...while` loop

### Ternary Operator
- Writing simple conditions in a single line using `?:`

### Numeric Formatting
- Formatting numbers as currency using `ToString("C")`
- Formatting output using placeholders (`{0}`, `{1}`)

### Parsing
- Using `int.TryParse()` to safely convert a string to an integer.
